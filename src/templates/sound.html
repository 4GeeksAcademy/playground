Fake Sound API

⚠️ The use of Postman or Insomnia (recomended) is strongly recomended for testing the API.


1. Get All Sound

            
              GET: /apis/fake/sound/all

              {
                fx: [],
                songs: []
              }

2. Get All Song FX

            
              GET: /apis/fake/sound/fx

              [
                {
                    id: 1,
                    category: "game",
                    name: "Game Over",
                    url: "files/mario/fx_gameover.wav"
                },
                {
                    id: 2,
                    category: "game",
                    name: "Jump Super",
                    url: "files/mario/fx_jump_super.wav"
                }
              ]

3. Get All Songs

            
              GET: /apis/fake/sound/songs

              [
                { 
                  "id":1, "category":"game", 
                  "name":"Mario Castle", 
                  "url":"files/mario/songs/castle.mp3" 
                },
                {
                  "id":2, 
                  "category":"game", 
                  "name":"Mario Star", 
                  "url":"files/mario/songs/hurry-starman.mp3"
                },
              ]

Here is an example of a song hosted on our servers:

https://playground.4geeks.com/apis/fake/sound/files/mario/songs/castle.mp3

You can see that all songs URL's start with https://playground.4geeks.com/apis/fake/sound/ and then are concatenated with the song path like this:

const songURL = "https://assets.breatheco.de/apis/sound/" + "files/mario/songs/castle.mp3";
